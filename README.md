# 🚀 GitHub Pages Deployment Test

This repository is a **technical sandbox** used to verify the deployment pipeline and hosting capabilities of **GitHub Pages**. 

## 🎯 Purpose
The goal of this project was to:
1.  Ensure the repository is correctly configured for static hosting.
2.  Test the integration of **HTML5**, **CSS3**, and **Vanilla JavaScript** in a live environment.
3.  Verify that **LocalStorage** functions correctly on a `github.io` domain.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables) and Flexbox.
- **JavaScript**: DOM manipulation and browser storage.
- **Deployment**: GitHub Pages (CI/CD).
